## ğŸ“ How It Works

Two simple formulas:

**1. Net Monthly Burn**
```
Burn = Monthly Expenses - Monthly Revenue
```

**2. Runway**
```
Runway (months) = Cash Balance / Net Burn
```

If you're profitable (revenue > expenses), your runway is infinite â™¾ï¸
